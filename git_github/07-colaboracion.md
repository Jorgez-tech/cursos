# 07. Colaboración Avanzada y Gestión de Proyectos

## ?? Issues, proyectos y discusiones
- **Issues**: Reporta bugs, solicita mejoras, documenta tareas
- **Proyectos**: Tableros tipo Kanban para organizar tareas
- **Discusiones**: Espacio para preguntas y propuestas

## ?? Organización de equipos
- Crea equipos en GitHub y asigna roles
- Usa repositorios privados y públicos según el caso

## ?? Gestión de tareas y milestones
- Asigna issues a miembros y milestones
- Usa etiquetas para categorizar tareas
- Haz seguimiento del progreso en el tablero de proyecto

## ?? Ejercicio Práctico
1. Crea un issue y asígnalo a un colaborador
2. Organiza tareas en un proyecto Kanban
3. Participa en una discusión técnica
