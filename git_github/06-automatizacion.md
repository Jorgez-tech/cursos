# 06. Automatización y DevOps con GitHub Actions

## ?? ¿Qué es GitHub Actions?
GitHub Actions permite automatizar flujos de trabajo como pruebas, despliegues y análisis de código directamente en GitHub.

## ??? Workflows y pipelines CI/CD
- Los workflows se definen en `.github/workflows/*.yml`
- Ejemplo básico:
```yaml
name: CI
on: [push]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Instalar dependencias
        run: npm install
      - name: Ejecutar pruebas
        run: npm test
```

## ?? Ejecución de pruebas automáticas
- Configura jobs para ejecutar tests en cada push o pull request
- Usa matrices para probar en múltiples versiones de Node, Python, etc.

## ?? Despliegue continuo
- Integra con servicios como Azure, AWS, Vercel
- Usa secrets para credenciales seguras

2. Agrega un job de despliegue (simulado)
3. Revisa el historial de ejecuciones en GitHub
